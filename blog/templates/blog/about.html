About Page

Hey everyone! This web application has been developed by Sarthak Saxena. I am a 2nd year engineering student at NIT, Jalandhar, India. You can find my GitHub link over here. I am an enthusiastic learner and developer, hoping to contribute to the community in whatever way I can.

This web app has been made entirely on the Django framework for python which manages the databases, dynamic pages, form validations, etc. I am the sole developer of this project. You can find the repo for this project over here.

There are several features on the app:
  • User Registration, Login with data validation
  • Posting, editing and deleting the blogs you have authored
  • Looking at all the blog posts by a particular author
  • Uploading a profile picture
  • Editing your details(email, username, profile picture)
  • Password reset via email
  • Logging out
  • Logging in to the admin server(if you have the access, that is :P)

...and several more on the way. This app is currently under development and will be updated every week with tons of exciting features.

If you wish to collaborate with me, feel free to contact me!
